    Layers Available

    New layer

    1. Create a repository on the project's gitlab and push data. Contact citiwatts@hevs.ch to ask access. Caution : datapackage.json is the key file for integration.
    2. The Integration file will copy data to geoserver
    3. Contact citiwatts@hevs.ch to define :
      • the legend
      • the content of the result panel
      • the category and the type of the layer
