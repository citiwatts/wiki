    Release 3.0.1/2 - 02.09.2025

    Fix

    • Display current value of slider in CM
    • 1000-separator export CSV
    • Feedback form
    • Description of inputs of type layer in CM

    Release 3.0.0 - 26.08.2025

    New functionalities

    • Responsive (work on mobile)
    • Calculation module sessions:
      • CMs can have multiple results
      • Results can be hidden
      • Results can be saved in account (if connected)
      • Results can be deleted locally and in account
    • PDF export of results (indicators et charts)
    • ZIP export of CM results
    • Charts data export in excel format to be able to recreate chart
    • Can expand session when logged in when log out timeout reached
    • Snapshot deletion validation
    • Calculation module order sorted by project

    New functionalities for CM developers

    • Dynamic inputs available
    • Date input available
    • Checkbox type input available
    • Order of CMs specific to each project (citiwatts, saphea, etc.)
    • CM developpers documentation improvement

    User experience improvements

    • Rework of Tools tab
      • Draggable tools bar
      • Selection data in result panel
    • Rework of Layer tab:
      • Personal layers now have legends, available directly below their title
      • Easier deletion and adding of personal layers
    • Rework of error messages:
      • Error messages popups do not disappear unless closed by user
      • The improvement of error messages are a work in progress (thank you for understanding)
    • Search feature is faster and limited by project (e.g. EU continent for EU projects)
    • Automatic fill of feedback form with user credentials if connected
    • Filename of .csv export improvement
    • Improvement of transparency in data status (loader)
    • Icon and color unification

    Bug fixes

    • Results possible below 1km2
    • 1 chart for layer "Electricity CO2 emissions - country average"
    • Snapshot feature stabilized

    Release 2.3.2 - 11.04.2025

    User experience improvements

    • "Stop CM" blinking button behaviour fixed
    • Improved management of canceling running CM
    • Map zoom and movement allowed when a CM is running, but unselect blocked
    • Found and fixed a backend issues that may have slowed down the platform (improved behaviour of platform for user)
    • Unification of layer categories (uppercase, between panels)
    • Version displayed in navbar, and more advanced version information displayed in "user" window (useful for user assistance)
    • In CMs tab, when a CM is selected, the title of the CM and the "go back" button now stay at the top of the tab

    Bug fixes

    • Import of inputs in CM
    • Links to standalone CM's wiki
    • Auto update of list of personal layers upon upload
    • Results of personal layers are removed when layer is unchecked
    • Added a security timeout on GDAL operations in the backend, to prevent potential resource leaks

    Release 2.3.0-06.03.2025 & 2.3.1-20.03.2025

    New functionalities

    • Sharing layer with selected email
    • Possibility of returning csv or zip files via CM in addition to the results displayed
    • More charts options for CMs results
    • Custom legends or no legend allowed for CM layer
    • Popup message to warn users about production release
    • Upload of personal geojson layers

    User experience improvements

    • Tooltip showing the available scales for running CMs
    • Improved list of layer when uploading a file, grouped by category
    • Order of calculation module
    • The data for a point of a personal layer of type industrial (excess heat, subsector and emissions) can be viewed by clicking on the map

    Bug fixes

    • Upload of geojson file at hectare level to select area on the map
    • Surface calculation redesigned for greater precision
    • Slider default value in calculation module
    • Column name when exporting result of layer
    • List of snapshot on small screen
    • Fix unit and value of mobility layers

    Release 2.2 - 28.11.2024

    New functionalities

    • Home page with project information
    • New login and registration system:
    • Better registration experience
    • User stays logged in between tabs
    • Satellite map set as default in order to better visualize the layers
    • Unselect all layers 
    • Update logos of partners and cookies information
    • Snapshots working with personal layers
    • Export CM inputs and results in a csv file
    • Import inputs (csv file) in CM in order to rerun with the same parameters
    • Search bar on CM tab

    New functionalities for CM developers

    • Mouse-over with information on input selection can be set by CM developers
    • Layer results can directly be saved in personal layers and reuse as input layer (if the CM developers configures it)
    • CM results can be text

    User experience improvements

    • Menu : auto close menu, synchronize checkbox of shared layer (if the shared layer selected is a personal layer)
    • "Files" popup window: % of usage, confirm deletion of personal layer
    • Add some tooltips
    • Better error messages for layer results

    Bug fixes

    • Region in Denmark in LAU2
    • Blurred layers: increased zoom level for calculation modules and personal layers
    • CM working at hectare level selection
